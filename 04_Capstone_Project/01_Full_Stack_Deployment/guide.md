# Full-Stack DevOps Capstone Project

## Project Overview
**Difficulty:** Advanced | **Time:** 12-15 hours | **AWS Cost:** ~$100/month

Complete production system demonstrating all DevOps skills integrated.

### What You'll Build

**Application Layer:**
- Python Flask REST API with PostgreSQL
- Containerized with Docker multi-stage builds
- Unit and integration tests with pytest
- Database migrations

**Infrastructure Layer (Terraform):**
- VPC with public/private subnets across 2 AZs
- ECS Fargate for serverless containers
- RDS PostgreSQL with encryption
- Application Load Balancer with SSL
- Auto Scaling based on metrics
- NAT Gateways for private subnet internet

**CI/CD Layer (GitHub Actions):**
- Automated testing on every PR
- Docker build and push to ECR
- Terraform plan on PRs
- Terraform apply on merges
- ECS blue/green deployments
- Rollback capabilities

**Monitoring Layer:**
- CloudWatch dashboards with custom metrics
- CPU, memory, and response time monitoring
- Alarms for high CPU, failed health checks
- SNS notifications to email/Slack
- Log aggregation and analysis
